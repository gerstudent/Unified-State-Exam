# Черепаха

## Задача №27

> Повтори 7 (Направо 90 Вперёд 4 Повтори 2 (Налево 90 Вперёд 4))

```python
from turtle import *

tracer(0)
r = 25

# Повтори
for i in range(7):
    rt(90)
    fd(4 * r)
    for j in range(2):
        lt(90)
        fd(4 * r)

up()

for x in range(-20, 20):
    for y in range(-20, 20):
        goto(x * r, y * r)
        dot(3, "blue")

update()
exitonclick()
```

# Чертежник

## Задача №9 (Сколько различных точек)

```python
from turtle import *

tracer(0)
screensize(3000, 3000)
r = 25


# сместиться
def move(a, b):
    return goto(xcor() + a * r, ycor() + b * r)


for i in range(7):
    move(6, -9)
    move(-6, 2)
    move(12, 3)

up()

for x in range(-20, 20):
    for y in range(-20, 20):
        goto(x * r, y * r)
        dot(3, "blue")

update()
exitonclick()
```
