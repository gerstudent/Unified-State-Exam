# Формулировки

## Задача 299 Поляков

> Складываются все цифры двоичной записи числа N. Если полученная сумма чётна, из числа убирают ведущую единицу (а также
> ставшие незначащими нули). В противном случае слева приписывается 1, а справа – два ноля.

```python
b = bin(n)[2:]
sm = b.count('1')
if sm % 2 == 0:
    b = b[1:].lstrip('0')
else:
    b = '1' + b + '00'
```
